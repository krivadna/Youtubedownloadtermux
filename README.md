
# Termux YouTube Downloader

An easy-to-use YouTube video downloader for Termux Android Users.

## One-Line Installation

```bash
wget https://raw.githubusercontent.com/krivadna/Youtubedownloadtermux/main/bin/ytdl2 -O $PREFIX/bin/ytdl2
```


## Usage

```bash
ytdl <YouTube URL>
```

Example:
```bash
ytdl https://youtube.com/watch?v=VIDEO_ID
```

## Features

- Download videos in best quality
- Download MP4 videos
- Extract MP3 audio
- View available formats
- Download custom formats
- Download playlists
- Auto-update yt-dlp

## Requirements

All requirements will be automatically installed:
- Python
- FFmpeg
- yt-dlp
