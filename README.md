## ⚡ One-Line Installation - Download the binary 

```bash
wget https://raw.githubusercontent.com/krivadna/Youtubedownloadtermux/main/bin/ytdl -O $PREFIX/bin/ytdl && chmod +x $PREFIX/bin/ytdl
```

## 🚀 Usage

```bash
ytdl <YouTube URL>
```

### Example:
```bash
ytdl https://youtube.com/watch?v=VIDEO_ID
```

## 🎯 Features

- 📹 **High Quality Video** - Download videos in best available quality
- 🎥 **MP4 Format** - Direct MP4 video downloads
- 🎵 **MP3 Extraction** - Convert videos to MP3 audio
- 📋 **Format Selection** - View and choose from available formats
- 📦 **Custom Quality** - Select specific video/audio formats
- 📂 **Playlist Support** - Download entire playlists
- 🔄 **Auto Updates** - Built-in yt-dlp updater
